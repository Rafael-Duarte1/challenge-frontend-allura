Amigo Secreto 

Este projeto consiste em um site simples para sortear nomes em um jogo de Amigo Secreto. Os usuários podem adicionar nomes e, ao final, realizar o sorteio para descobrir quem será o amigo secreto de cada participante.

 Funcionalidades

Adicionar nomes à lista.

Sortear um nome aleatório entre os participantes.

Interface simples e intuitiva.
